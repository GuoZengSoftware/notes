## Gradle

* 下载
* https://gradle.org/post-download-gradle/
* 配置环境变量(前提需Java环境变量)
```
GRADLE_HOME=Gradle根目录
%GRADLE_HOME%/bin=添加到Path环境变量中($GRADLE_HOME/bin --Linux或Mac下的环境变量) Linux可以在 ~./bashrc文件中配置
```

* 配置完检查版本
```
gradle -v
```

* jar包存放目录 修改完需重启电脑生效
```
GRADLE_USER_HOME=E:\Library\Gradle\.gradle
```
